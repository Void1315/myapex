# My-Apex
> 用于apex一键更新cfg，开启sg、一键拾取、云同步自己的cfg配置等场景

## dev启动

```bash
git clone git@github.com:Void1315/myapex.git

yarn

yarn run dev
```

## 调试

使用`vscode`进行调试，已配置好`launch.json`、`tasks.json`文件，请运行`Debug: All`配置即可。
